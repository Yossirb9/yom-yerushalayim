# יום ירושלים — מסע בעיר הקודש

משחק הרפתקאות חינוכי בסגנון Pokémon GBC על יום ירושלים, מותאם לתלמידי כיתות ד׳-ו׳.

לבית הספר "צליל למצוינות" ע״ש לילה חלילי.
פותח ע"י יוסי רבינוביץ׳.

🌐 **שחק במשחק:** [yom-yerushalayim.vercel.app](https://yom-yerushalayim.vercel.app)

## עלילה

ארבעה תלמידי בית ספר יסודי יוצאים לטיול ירושלים ביום ירושלים. הם נפגשים עם דמויות לאורך 5 תחנות בעיר העתיקה ולומדים עובדות אמיתיות על ההיסטוריה והתרבות של ירושלים. בסוף היום הם חוזרים לבית הספר ומציגים את מה שלמדו בטקס.

## דמויות שחקן

- **דניאל** — אוהב ספורט. הולך לטפס על כל קומה במצודת דוד.
- **שירה** — אוהבת לשיר. רוצה לבצע ב"ירושלים של זהב" בטקס.
- **אורי** — חובב היסטוריה. רושם 5 עובדות חדשות בפנקס שלו.
- **יעל** — אוהבת אוכל. כותבת ביקורת לעיתון בית הספר על 4 מאכלים.

## תחנות

1. **מצודת דוד** — ארכיאולוג מספר על שכבות הבנייה והשם המטעה
2. **הכותל המערבי** — רב מסביר שזה קיר התומך של הר הבית. אפשר לכתוב פתק.
3. **שוק מחנה יהודה** — מוכר מציג את השוק וטועם פיתה
4. **טיילת הס** — חיילת מצביעה על הר הזיתים והר הצופים
5. **מרכז העיר** — מדריך מציג את סמל ירושלים. גם תחנת רכבת קלה.

## טכנולוגיה

- **React 18** + **Vite** + **TypeScript**
- **Tailwind CSS** עם RTL מלא
- **Zustand** ל-state + localStorage persist
- **HTML5 Canvas** עם רינדור פיקסל-ארט וספרייטים מ-PNG sheets
- **Vercel** לפריסה

## פיתוח מקומי

```bash
npm install
npm run dev        # ב-http://localhost:5173
npm run build      # פרודקשן ב-dist/
npm run preview    # בדיקה מקומית של גרסת פרודקשן
```

## מבנה התיקיות

```
src/
  engine/          מנוע ליבה: canvas, tiles, sprites, input, dialog
  game/
    maps/          5 מפות: school, jerusalem, ceremony + 3 פנים-בתים
    npcs/          הגדרות 4 תלמידי שחקן
    interactives/  כל הדיאלוגים והאינטראקטיביים
    quests/        רישום המשימות (3 משימות)
    dialogs/       פיצוץ דיאלוגים פר דמות+מיקום
    story/         טריגרים על דגלי סיפור
  store/           Zustand store
  ui/
    screens/       Title, CharacterSelect, Game, Menu, Ending
    hud/           DialogBox, QuestHud, Clock, TouchControls
public/
  sprites/         PNG sprite sheets (8x8 tileset, 4x4 character)
```
